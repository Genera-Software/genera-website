6 hooks: prediction · insider · lab rat · expert · mirror · sledgehammer (provocative, rationed).
Every piece passes this gate before you film or build:
  • Which hook type, or two stacked, opens it?
  • What is the open loop, and where does it pay off (not in the first 3 seconds)?
  • What story carries it, not a feature list?
  • One feeling you are aiming for: anger, relief, fear, joy, tension?
  • Is every line pulling its weight as copy?
  • How is delivery briefed: pace, pauses, the payoff beat?

Refresh analytics from Instagram

How the live numbers get in.
This dashboard is a local file, so it cannot call Instagram itself. A small script pulls the numbers and writes a file you load here.
Free route (recommended): Meta Graph API. One-time token setup, then in this folder run
IG_TOKEN=your_token node ig-fetch.js
It writes ig-posts-DATE.json. Click Restore backup and pick that file. Your ideas and to-dos are kept. See SETUP.md for the token steps.

Per-post views, reach, saves, comments and reel watch time fill automatically. Per-post gender is not in the API, so the male % stays the figure you read in the app. Paid alternative: renew Windsor and ask Claude to pull.
